# Roll-a-Ball game

A simple beginner-friendly game developed in Unity using C#. This project includes a survival-based gameplay twist with countdown timer.

# Gameplay

* Controlling a ball using the keyboard.
* A "60-second countdown timer" begins when the game starts.
* Our objective is to keep the ball on the platform until the timer reaches "00:00".
* If we successfully survive for the full minute, the game displays "YOU WON!".
* If the ball falls off the platform before the timer ends, the game displays "YOU LOST!".

# Features

* Smooth ball movement using Unity physics.
* 60-second countdown timer.
* Win and lose conditions.
* Simple UI for timer and game result messages.
* Beginner-friendly project structure.

# Technologies Used

* Game Engine: Unity
* Programming Language: C#
* IDE: Visual Studio

# Controls

* W / A / S / D or Arrow Keys – Move the ball.

# Purpose

This project was created to practice the fundamentals of Unity game development, including player movement, physics, UI elements, timers, and game state management.
